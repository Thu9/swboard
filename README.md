# 댓글 게시판(reply Bulletin Board System) - 개인 프로젝트 

## 목적
- 신입 웹 개발자의 기본소양
- 어떠한 웹이던지 게시판은 필수요소
- 반복된 연습으로 빠르고 최적화된 게시판 생성을 목표로함
--------------------------------------------------------

## 개발환경
- Spring 3.9
- JDK 11.0
- oracle 12.1
- Mybatis 3.2
- Tomcat 8
